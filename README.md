# 🚆 Railway Reservation System (Java + MySQL)

## 📌 Description
A console-based Railway Reservation System developed using Java and MySQL. The system allows users to search trains, book tickets, cancel bookings, manage waiting lists, and automatically confirm tickets when seats become available.

## 🚀 Features
- 🔍 Search trains by source
- 🎫 Book tickets
- ❌ Cancel tickets
- 📋 View bookings
- ⏳ Waiting list system
- ✅ Auto seat confirmation

## 🛠 Technologies Used
- Java
- JDBC
- MySQL

## ▶️ How to Run
1. Import the database using `railway_db.sql`
2. Open project in Eclipse
3. Add MySQL Connector JAR
4. Run `TestDB.java`
